# wiki/ — LLM 컴파일 위키

이 폴더는 Claude가 `raw/` 를 읽고 합성·유지하는 공간입니다. **사람이 직접 편집하지 마세요.**

## 폴더 구조

| 폴더 | 내용 |
|------|------|
| `architecture/` | 시스템 구조도, 기술 선택 근거, 컴포넌트 관계 |
| `troubleshooting/` | 증상→원인→해결 형식의 트러블슈팅 가이드 |
| `devices/` | 디바이스 인벤토리, 현황, 알려진 이슈 |
| `operations/` | 펌웨어 배포, 서비스 재시작, 백업/롤백 절차 |
| `index.md` | 전체 위키 목차 |
| `log.md` | 컴파일 이력 |

## 항목 작성 규칙

**트러블슈팅 항목 템플릿:**
```
## [증상 한 줄 요약]
### 증상
### 원인
### 해결
### 재발 방지
> 출처: raw/...
```

**병합 원칙:** 기존 항목을 삭제하지 않고 새 정보를 추가합니다. 상충 시 날짜로 구분하여 양쪽 기록.

**출처 표기:** 모든 항목 하단에 `> 출처: raw/폴더/파일명.md` 형식으로 명시.
